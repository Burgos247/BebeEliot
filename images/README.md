# 📸 Fotos de la decoración

Coloca aquí las fotos del baby shower para que aparezcan en la sección **"Nuestra decoración"** de la página.

Nómbralas exactamente así:

| Archivo       | Sugerencia de foto                          |
|---------------|---------------------------------------------|
| `photo1.jpg`  | Mesa principal con el arco de globos         |
| `photo2.jpg`  | Centro de mesa: osito con globo de confeti   |
| `photo3.jpg`  | Arco de globos azul y blanco                 |
| `photo4.jpg`  | Nube de globos con lucecitas                 |
| `photo5.jpg`  | Ilustración del osito de Eliot               |
| `photo6.jpg`  | Otra foto de la decoración                   |

> Si una foto no existe, simplemente no se muestra (no rompe la página).
> Puedes usar `.jpg`, o cambiar las rutas en `index.html` si prefieres otro formato.
